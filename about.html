O aplikaciji

StudyNotes je jednostavna web aplikacija razvijena u edukativne svrhe. Cilj aplikacije je demonstracija osnovnih principa rada sa HTML-om, CSS-om, JavaScript-om, jQuery-jem i AJAX tehnologijom.

Aplikacija koristi dinamičko učitavanje sadržaja bez osvežavanja stranice, čime se poboljšava korisničko iskustvo.

  • HTML i CSS – struktura i stilizacija
  • JavaScript – osnovna logika aplikacije
  • jQuery – manipulacija DOM elementima
  • AJAX – dinamičko učitavanje sadržaja
